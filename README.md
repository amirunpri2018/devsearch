# devsearch
Google for devs.

## Stack
- Python 3
- MySQL
- Flask
- Bootstrap 4
- Scrapy

## Setup
- Clone or download this repo
- Create and/or activate a virtual environment
- Run `pip install -e .`
- Create MySQL db
- Create a *.env* file from *.env.example*
- Run `flask db upgrade`
- Run `flask run`
